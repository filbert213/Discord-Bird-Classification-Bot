# 🐦 Discord Bird Classification Bot

Bot Discord berbasis AI yang dapat mengidentifikasi jenis burung dari gambar yang dikirim oleh pengguna. Model image classification dilatih menggunakan Google Teachable Machine untuk mengenali tiga jenis burung: **Merpati**, **Ayam**, dan **Bebek**.

---

## 📖 Deskripsi

Discord Bird Classification Bot memanfaatkan model machine learning yang telah dilatih menggunakan Google Teachable Machine. Pengguna cukup mengirimkan gambar burung ke bot melalui Discord, kemudian bot akan memproses gambar tersebut dan memprediksi apakah burung tersebut adalah **Merpati**, **Ayam**, atau **Bebek**.

Setelah proses klasifikasi selesai, bot akan mengirimkan hasil prediksi beserta **confidence score** (tingkat kepercayaan) sehingga pengguna dapat mengetahui seberapa yakin model terhadap hasil tersebut.

---

## ✨ Fitur

- 🐦 Mengidentifikasi gambar burung.
- 🤖 Menggunakan model AI dari Google Teachable Machine.
- 📷 Menerima gambar langsung melalui Discord.
- 📊 Menampilkan hasil klasifikasi beserta confidence score.
- ⚡ Memberikan hasil dalam beberapa detik.

---

## 🛠️ Teknologi

- Python
- Discord.py
- TensorFlow / Teachable Machine
- Image Classification
- Discord API

---

## 🚀 Cara Menggunakan

1. Jalankan bot Discord.
2. Kirim gambar burung ke channel yang telah ditentukan.
3. Bot akan memproses gambar menggunakan model AI.
4. Hasil klasifikasi akan ditampilkan, misalnya:
   - Merpati (96.4%)
   - Ayam (91.7%)
   - Bebek (98.2%)
